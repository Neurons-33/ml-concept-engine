# Linux 指令知識庫

## 建立 Python 虛擬環境
用途：
建立 Python 專案虛擬環境

指令：
```bash
python3 -m venv venv
```

## 啟動 Python 虛擬環境
用途：
啟動虛擬環境，使 Python 套件安裝在專案中

指令：
```bash
source venv/bin/activate
```

## 離開 Python 虛擬環境
用途：
退出虛擬環境

指令：
```bash
deactivate
```

## 查看目前資料夾位置
用途：
顯示目前所在的資料夾路徑

指令：
```bash
pwd
```

## 查看 Python 版本
用途：
確認目前使用的 Python 版本

指令：
```bash
python --version
```

# Linux 資料夾操作

## 查看資料夾內容
用途：
列出資料夾中的檔案

指令：
```bash
ls
```

## 查看資料夾詳細資訊
用途：
顯示檔案權限、大小、時間

指令：
```bash
ls -l
```

## 顯示隱藏檔案
用途：
顯示隱藏檔案（例如 .git）

指令：
```bash
ls -a
```

## 進入資料夾
用途：
切換到指定資料夾

指令：
```bash
cd 資料夾名稱
```

## 回到上一層資料夾
用途：
回到父資料夾

指令：
```bash
cd ..
```

## 回到 home 資料夾
用途：
回到使用者主目錄

指令：
```bash
cd ~
```

## 建立資料夾
用途：
建立新的資料夾

指令：
```bash
mkdir 資料夾名稱
```

## 建立多層資料夾
用途：
一次建立多層目錄

指令：
```bash
mkdir -p folder/subfolder
```

## 刪除資料夾
用途：
刪除資料夾及其內容

指令：
```bash
rm -r 資料夾名稱
```

# Linux 搜尋工具

## 搜尋檔案中的文字
用途：
在檔案中搜尋指定文字

指令：
```bash
grep "Hello" file
```

## 搜尋資料夾中的關鍵字
用途：
在資料夾所有檔案中搜尋關鍵字

指令：
```bash
grep -r "print" .
```

## 搜尋並顯示行號
用途：
顯示關鍵字所在行號

指令：
```bash
grep -n "print" file
```

## 搜尋忽略大小寫
用途：
忽略大小寫搜尋

指令：
```bash
grep -i "Hello" file
```

## 搜尋資料夾並顯示行號
用途：
遞迴搜尋並顯示行號

指令：
```bash
grep -rn "print" .
```

## 使用 pipe 搜尋文字
用途：
透過管道搜尋文字

指令：
```bash
cat file | grep print
```

## 搜尋專案中的 Python 檔案
用途：
找到專案所有 .py 檔案

指令：
```bash
find . -name "*.py"
```

# 標準專案結構

## 建立標準 ML / NLP 專案資料夾
指令：
```bash
mkdir -p src data/{raw,processed,external}
```

## data 資料夾用途
raw：原始資料  
processed：清理後資料  
external：外部資料來源

## src 程式碼結構
src 包含：

__init__.py → Python package  
main.py → 程式入口  
preprocess.py → 資料清理  
tokenizer.py → tokenization  
model.py → NLP architecture  
pipeline.py → 串聯整體流程
